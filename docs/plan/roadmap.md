# SmartPenny Future Roadmap 🚀
*Last Updated: 2026-01-16*

## 1. 🛠️ 설정 및 유지보수 (Settings & Maintenance)

### 1.1 프리셋 시스템 (Preset System)
*   **가져오기 매핑 저장**: 사용자가 엑셀 가져오기 시 설정한 '컬럼 매핑 규칙'을 은행별/카드사별로 저장.
*   **자동 적용**: 파일명에 '신한카드'가 포함되면 자동으로 '신한카드 프리셋' 적용.

### 1.2 데이터 백업 및 복원 (Backup & Restore)
*   **JSON 내보내기/가져오기**: 전체 데이터(자산, 거래, 설정)를 JSON 파일로 백업.
*   **클라우드 스냅샷**: 구글 드라이브 등에 주기적으로 자동 백업.

---

## 2. ⚡️ 성능 최적화 (Performance Tuning)

### 2.1 대량 데이터 처리
*   **가상화 리스트 고도화**: `react-virtuoso` 최적화를 통해 1만 건 이상의 거래 내역 스크롤 성능 개선.
*   **검색 인덱싱**: `Fuse.js` 인덱스를 웹 워커(Web Worker)로 분리하여 메인 스레드 부하 감소.

### 2.2 앱 초기 로딩
*   **코드 스플리팅 (Code Splitting)**: 라우트 기반의 Lazy Loading 적용.
*   **이미지 최적화**: WebP 포맷 적극 도입 및 이미지 크기 최적화.

---

## 3. 📊 고급 분석 (Advanced Analytics)

### 3.1 자산 변동 리포트
*   ** 순자산 흐름**: 월별 자산/부채/순자산 변화 추이 그래프.
*   **지출 패턴**: 요일별/시간대별 주요 지출 패턴 분석.

### 3.2 예산 고도화
*   **이월 예산 (Rollover Budgeting)**: 남은 예산을 다음 달로 이월하는 옵션.
*   **카테고리 그룹핑**: 상위/하위 카테고리 계층 구조 도입.

---

## 4. 📱 모바일 경험 (Mobile Experience)

### 4.1 PWA (Progressive Web App)
*   **설치형 앱 경험**: 홈 화면 추가 아이콘 및 스플래시 스크린.
*   **오프라인 모드**: 네트워크 연결 끊김 시 로컬 데이터로 읽기/쓰기 후 동기화.

### 4.2 위젯 (Widgets)
*   **iOS/Android 위젯**: 바탕화면에서 잔액 및 최근 지출 확인.
